# Pause YouTube / Resume YouTube When Other Tabs Play Audio

This browser extension automatically **pauses YouTube videos** when audio starts playing in any other browser tab and **resumes playback** once the external audio stops.

Designed for Chromium-based browsers such as **Brave** and **Google Chrome**.

---

## Features

- Automatically pauses YouTube when another tab plays audio
- Automatically resumes YouTube when external audio stops
- Does not interfere if the user pauses YouTube manually
- Lightweight and no external dependencies
- Works fully offline

---

## Installation (Developer Mode)

1. Open your browser and go to:

```bash
brave://extensions
```

2. Activate "Developer Options"

3. "Load extension without packing"

4. Select the repository folder and try it out
